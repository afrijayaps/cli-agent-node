# Agent Notes

File ini berisi aturan kerja singkat untuk agent/kontributor otomatis.

## Task List Wajib

Setiap perubahan non-trivial harus dicatat di `task-list.md`:

- Tambah satu baris task baru jika belum ada.
- Update kolom `status`, `updated_at`, dan `notes` jika ada progres.
- Gunakan format tanggal ISO: `YYYY-MM-DD`.
- Jika `blocked`, jelaskan penyebabnya di `notes`.

Tujuan: menjaga jejak pekerjaan lintas agent tetap jelas dan rapi.
